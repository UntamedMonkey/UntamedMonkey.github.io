STL简介:

C++的标准模板类库，包含常见数据结构及函数。主要包括:

  • 容器
  • 算法
  • 函数（Functions）
  • 迭代器

1. 容器：

  • 序列式容器:基本的容器
    • arrays||at();[];front();back();size();empty();fill()||比C自带的数组好
    • vector||asign();at();[];front();back();begin();end;size();max_size();capacity();resize(n);empty();shrink_to_fit();push_back();pop_back();insert();erase();clear();
    • forward_list||assign();push_front();emplace_front();pop_front();insert_after();emplace_after(); erase_after(); remove(); remove_if();clear();
    • deque
    • list
  • 容器适配器:序列式容器的扩展
    • queue
    • priority_queue
    • stack
  • 关联式容器:非序列式容器，基于自平衡二叉树
    • set
    • multiset
    • map
    • multimap
  • 无序关联式容器:非序列式容器，基于哈希表
    • unordered_set
    • unordered_multiset
    • unordered_map
    • unordered_multimap

  • sort(startaddress, endaddress)
